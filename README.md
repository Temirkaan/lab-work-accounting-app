# Система учёта выполненных лабораторных работ

Учебное Flask-приложение для темы 19.

## Возможности

- добавление студентов;
- добавление лабораторных работ;
- фиксация сдачи работы;
- проверка и выставление оценки;
- отправка работы на доработку;
- JSON API для списка сдач.

## Запуск

```powershell
pip install -r requirements.txt
python app.py
```

Сайт откроется по адресу `http://127.0.0.1:5000/`.

## Документация

Отчёт по заданиям 2-4 находится в папке `docs/`.
